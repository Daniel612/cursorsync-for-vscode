## Installation Guide
- **VSCode Marketplace**
  1. Search for "CursorSync for VSCode" in the VSCode Extensions Marketplace.
  2. Click "Install".

- **Local Installation**
  1. Download the latest `.vsix` package.
  2. In VSCode/Cursor/Trae, go to Extensions.
  3. Drag and drop the `.vsix` file.
## Requirements
- Xcode
